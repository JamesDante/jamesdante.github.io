均值，方差，标准差三个常见的统计学概念。 但是他们在另外一个数学工具条件下的计算方法完全不同(但意义是相似的)。

这个数学工具叫**随机变量**，维基百科上面对它的解释是“用来表示随机试验结果的变量”，听起上去很抽象但其实非常简单

例如扔硬币：正面的概率 1/2， 反面的概率1/2。
这里正面和反面的表达不够数学，于是我们用0和1来代替正面和反面，我们就得到了一个数学意义上的随机变量：

```
扔硬币

0：1/2
1：1/2
```

**均值：**  
在一组数据中就很简单，就是这一组数据的和除以数量得到的平均值。
```
{1，3，2，2}

(1 + 3 + 2 + 2)/4 = 2
```
就是 $\sum x \div n$

在随机变量的环境里，均值就是求期望，求期望就是概率乘上对应值的和，还是用丢硬币的例子：

期望 = $0 \times 1/2 + 1 \times 1/2 = 1/2$

可以简写成 $\mu = \sum xp$


**方差：**  
对于一组数据来说方差的步骤就是： 
1. 求这组数值的平均值。
2. 然后用每个数值去减去平均值的平方。
3. 然后得到的各个数值的结果再求平均值。

还是这组数据：
```
{1，3，2，2}
```
(1 + 3 + 2 + 2)/4 = 2

$((1-2)^2 + (3-2)^2 + (2-2)^2 + (2-2)^2)/4  = 0.5$


对随机变量：
1. 跟求期望的方式基本一样，唯一就是把数值平方再乘上概率。
可以简写成 $\sum x^2p$
2. 减去期望的平方。 

最后就是 $\sum x^2p - \mu^2$ 

```
还是来扔硬币

0：1/2
1：1/2
```
$(0^2\times 1/2 + 1^2\times 1/2) - (1/2)^2 = 1/4$  


**标准差：**  
这个值直接对他们的方差求平方根就行了。
