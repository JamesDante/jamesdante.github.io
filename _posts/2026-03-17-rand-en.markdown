Mean, variance, and standard deviation are three common statistical concepts. But under a different mathematical framework, their calculations work quite differently — though the underlying meaning stays similar.

That framework is the **random variable**. Wikipedia defines it as "a variable used to represent the outcome of a random experiment" — sounds abstract, but it's actually very simple.

Example: flipping a coin. Heads probability = 1/2, tails probability = 1/2. "Heads" and "tails" aren't very mathematical, so we replace them with 0 and 1. We now have a random variable in the mathematical sense:

```
Coin flip

0: 1/2
1: 1/2
```

**Mean:**

For a dataset, this is straightforward: sum all values and divide by the count.

```
{1, 3, 2, 2}

(1 + 3 + 2 + 2) / 4 = 2
```

Written as $\sum x \div n$.

For a random variable, the mean is the **expected value**: multiply each value by its probability and sum the results. Using the coin flip:

$E = 0 \times 1/2 + 1 \times 1/2 = 1/2$

Compactly: $\mu = \sum xp$

**Variance:**

For a dataset, the steps are:
1. Compute the mean.
2. Subtract the mean from each value and square the result.
3. Average all those squared differences.

Using the same dataset:
```
{1, 3, 2, 2}
```

$(( 1-2)^2 + (3-2)^2 + (2-2)^2 + (2-2)^2) / 4 = 0.5$

For a random variable:
1. Similar to computing expectation — square each value, then multiply by its probability. Written as $\sum x^2 p$.
2. Subtract the square of the expected value.

Result: $\sum x^2 p - \mu^2$

```
Coin flip again:

0: 1/2
1: 1/2
```

$(0^2 \times 1/2 + 1^2 \times 1/2) - (1/2)^2 = 1/4$

**Standard Deviation:**

Simply take the square root of the variance.
